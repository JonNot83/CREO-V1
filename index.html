Enabling Creo Superpowers

1. Copy/Paste the below table to your config.pro:

!Pro/Web.link Options Setting Description
web_enable_javascript on !Enables Pro/Web.link
web_link_proe_read yes !Lets web.link read from the CREO sessio
web_link_proe_write yes !Lets web.link write to the CREO session
web_link_file_read yes !Lets web.link write to the file system
web_link_file_write yes !Lets web.link read the file system

2. Go to your Internet Explorer Options > Security > Trusted Sites and add this site.

 Under "Custom Level" change the following:
 Prompt "Download unsigned ActiveX controls"
 Enable "Initialize and Script ActiveX modules"

3. Use the items below in the Pro-E integrated browser.
